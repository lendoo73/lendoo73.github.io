Technology Wiki

Logic Gate

An idealized or physical device that performs a basic logical operation on one or more binary inputs and produces a binary output.

ASCII

As English name is American Standard Code for Information Interchange. Computers only understand numbers, so it describes the calculated values ​​needed to interpret the characters 'a' or '@'.

RAM

A random access read / write storage device. Stores the programs to be executed by the CPU and the data to be processed. The speed of access to a particular piece of memory is independent of its location.

Programming language

A series of instructions used in computing that can be read and interpreted by man to communicate directly or indirectly (for example, after translation into machine code) to a computer how to perform a particular task.

DNS

Domain Name System. A hierarchical, highly decentralized naming system for computers, services, and any resource connected to the Internet or a private network.

HTTP

Hyper Text Transfer Protocol. An application protocol for distributed, collaborative and hypermedia information systems. The basis of data communication on the World Wide Web. This is the basic protocol for exchanging and moving hypertext.
